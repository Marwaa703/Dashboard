# Dashboard Project

A full-stack dashboard application with React frontend and Node.js backend.

## Project Structure

- `/client` - React frontend application
- `/server` - Node.js backend application

## Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

## Getting Started

### Backend Setup

```bash
cd Server
npm install
nodemon index.js
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```
